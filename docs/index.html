Genetic Mutation Insight System

1. Problem Statement

Genetic mutations are responsible for many inherited disorders. This project aims to build a system that stores, analyzes, and predicts disorder risk based on mutation data.

2. Motivation

Existing bioinformatics tools are complex and require expertise. This project aims to build a simplified educational system using Python and SQL.

3. Literature Review

  • Paper 1: Mutation-based disease prediction using ML
  • Paper 2: Computational analysis of DNA mutations
  • Paper 3: Bioinformatics approaches in genetic disorder detection

4. Dataset

Dataset sourced from Kaggle containing mutation type, gene name, mutation position and disorder classification.

5. Methodology

  • Data Cleaning
  • Database Design (SQL)
  • Python Analysis (GC content, frequency)
  • Rule-Based Risk Prediction
  • Future Scope: ML Integration
